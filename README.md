# Sample ENV 
 
## For client
```
BASE_URL="http://localhost:8003"
REDIRECT_URL="http://localhost:4000/"
```

## For server
```
DATABASE_URL="mongodb+srv://<username>:<password>@<cluster-name>.mongodb.net/?retryWrites=true&w=majority"

```

#  To run the project

## For client
```
cd client
npm install
npm start
```

## For server
```
cd server
pip install -r requirements.txt
python main.py
```

## For **stores**
```
cd stores
pip install -r requirements.txt
python main.py
```
